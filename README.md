# katakana-eigo
英文をカタカナ英語に変換します．

## 例

有名な例文です．

```text
The quick brown fox jumps over the lazy dog.
```

```text
ザ クウィク ブラウン ファクス ジャンプス オウヴァー ザ レイズィー ダグ
```

```text
In such cases, no phonetic vowel is pronounced between the last two consonants, and the last consonant forms a syllable on its own.
```
